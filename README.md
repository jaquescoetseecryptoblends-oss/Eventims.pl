# Eventims.pl
# Event¡m Web 3.0 Ticketing Site

This is a **Web 3.0 ticketing website** for live concerts and events, inspired by Eventim.  

## Features
- Displays upcoming concerts with **images/videos**, **dates**, **times**, and **venues**
- Seat-based **ticket pricing** (e.g., Enrique Iglesias 150$–1500$)
- **Crypto payment integration**:
  - Bitcoin (BTC)
  - USDT (TRC-20, Tron)
  - BNB (Binance Smart Chain)
- **MetaMask / Web3 integration** for smart contract payments
- Admin panel to **add or edit events**
- Search and filter functionality by name or price

## How to Use
1. Clone or download this repository
2. Open `index.html` in your browser to view the site
3. Use the “Buy” button on an event to pay via crypto or Web3
4. Admin panel allows updating events dynamically

## Technologies
- HTML, CSS, JavaScript
- Ethers.js for Web3 / smart contract integration
- QRCode.js for crypto QR codes
